# Agent instructions for Agile Fragile

## Copy style (permanent)

**Never use em dashes (—) in site copy, page titles, meta descriptions, or the downloadable model specification.**

Use instead:

- Commas for light pauses and contrast
- Colons for labels and elaboration
- Periods to split ideas into separate sentences
- Parentheses for asides

The regression test (`node scripts/regression-test.mjs`) enforces this on all HTML pages and `downloads/human-agile-model-specification.md`.

## Merge policy

Merge pull requests when work is complete. Do not ask the user for merge confirmation.
